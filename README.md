# Password Generator

## Description
Un petit projet Python pour générer des mots de passe aléatoires selon les critères choisis par l'utilisateur.

## Fonctionnalités
- Lettres majuscules et minuscules
- Chiffres
- Caractères spéciaux
- Longueur minimum configurable
